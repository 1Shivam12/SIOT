Twitter API: Twython

A python wrapper, twython, for twitter was used to stream data from the official twitter API. The API call that was made initiated a twython streamer class to start search for tweets containing a few key words that correspond to tweets about movies, these were

  1. Movies
  2. Cinema
  3. Film
  4. Vue Cinema
  5. Cineworld

The number of tweets collected per day were pretty much the same throughout the data collection period. It can therefore be seen that there isn't much of a data variation with time in this data set.
