The Assignment

As part of 4th Year in Design Engineering at Imperial College London I am undertaking a module called Sensing & Internet of Things (SIOT). The main requirements of the project were

  1. Acquire sensors and other data at the appropriate sampling rate
  2. Establish the Communications and Networking protocols for data storage
  3. Provide data Storage and/or Publishing via APIs
  4. Streamlined basic time-series Data Analysis
  5. An App
  6. Data Cross correlation/fusion, complementing data, novel interactions, control and actuation
  7. Present and discuss opportunities Innovation, creativity, enterprise, scalability & complexity analysis within your ecosystem

The Web App

The W2W app takes information from Twitter, Weather APIs and IMDB to produce a threshold value that is examined against a function value to assign a score to each film title listed on the website. Based on these function values, it suggests weather or not to watch a given movie based on the existing state of the data.
